# PROJECT MEMORY & STATE

- We are building a 10-module architecture.
- **Current Phase:** Phase 1 (Foundation & Login/Auth Setup).
- Do not build features for Phase 4 (Voice) or Phase 6 (Analytics) if we are currently working on Phase 1 or 2.
- Always assume the target hardware is a low-end Android device with a prepaid, restricted data plan. Heavy processing must be deferred to the cloud, and heavy payloads must be minimized.
