# WRENLY AI - CORE CONTEXT

## The Problem
The Philippine basic education sector faces a severe learning crisis (low PISA scores, functional illiteracy, overcrowded classrooms, teacher burnout, and poor internet infrastructure).

## The Solution
Wrenly AI is an offline-first, multilingual AI classroom app for Philippine public schools. It converts complex teacher materials into simplified, voice-enabled, localized lessons.

## Core Mandates
1. **Offline-First:** The app MUST function without an internet connection. Data is saved to local SQLite and synced to the cloud in the background when connectivity returns.
2. **The Vocal Bridge:** Reading comprehension is bypassed using localized Text-to-Speech (TTS). Audio is prioritized over text.
3. **Multilingual:** Strict support for English, Tagalog, and Bisaya.
4. **Co-Pilot, Not Autopilot:** AI assists teachers (summaries, quizzes) but requires human-in-the-loop approval.
